## 1、项目介绍

基于springboot的校园物流管理系统1拥有两种角色

管理员：物流管理（揽件、寄出）、用户管理等

用户：收件、寄件、个人物流信息管理等


## 2、项目技术

后端框架：springboot、Mybatis

前端技术：Layui、html、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.5-5.7、8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows
